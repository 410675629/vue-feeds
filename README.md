# vue-seeds

## download

``` shell
git clone git@github.com:leinue/vue-feeds.git
```
## run

使用8080端口，注意8080端口别被占用

``` shell
npm install
npm run dev
```
or

``` shell
npm install
webpack
```
or
``` shell
npm install
webpack-dev-server
```
